# Public Assets

Vite 自动从这里 serve 文件到 site 根路径。

## Hero 图片

把任意一张氛围照片命名为 **`hero.jpg`** 放到这个文件夹，editorial 版 Dashboard 顶部就会自动接上。

**推荐尺寸**：1600×400 或更大，横向构图。

**素材来源选项**：
1. 直接保存 perplexity.ai/comet 页面上的草甸/玻璃球图（团队内部使用）
2. Unsplash 搜索 `wildflower meadow` / `glass sphere meadow` / `morning light field`
3. 任何你觉得对味的暖光自然摄影

如果不放图片，hero 区会显示 CSS 渐变作为 fallback。
